This is an investigative project I developed in order to identify the best performing Regression model 
among three chosen ones. These being:
->LinearRegression() /n
->Lasso() /n
->Ridge() /n
The models were all trained using the same Synthetic Sales data from a downloaded Kaggle dataset. 
The evaluation was done by calculating the Mean Absolute Error of the data's test set, to identify
if it surpasses/improves from the baseline MAE. The investigation follows an ML workflow of:
1. Prepare Data /n
2. Build Model /n
3. Evaluation/Communicate Results
